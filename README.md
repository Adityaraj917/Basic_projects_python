#PYTHON PROJECTS

This repository contains a collection of projects from beginner level to intermadiate level which helps me to build the foundational level of programming.  
Each script focuses on a specific concept such as automation, AI assistance, API handling, games, text processing, and more.


  **Projects Included**

1. AI_assistant.py  
A simple voice-based personal assistant that listens to user commands and performs tasks like opening apps, searching the web, etc.

2. ATM.py  
A console-based ATM simulation supporting deposit, withdrawal, balance check, and PIN authentication.

3. File_transfer.py  
Automates file transfer between directories using Python scripting.

4. Movie_ticket_permission.py  
A logic-based program that checks if a person is eligible to watch a movie based on age and permissions.

5. Tic_tae_toe_game.py  
A classic Tic-Tac-Toe game built using Python. Two-player mode with win/lose/draw detection.

6. calci.py  
A basic calculator supporting arithmetic operations like addition, subtraction, multiplication, and division.

7. chatbot.py  
A rule-based conversational chatbot that responds to predefined user inputs.

8. corporate_simple.py  
A simple corporate employee record system storing & displaying employee details.

9. email_automation.py  
Automates the process of sending emails using SMTP. Includes subject, recipient, and message content support.

10. flask_api.py  
A basic Flask REST API that demonstrates API creation, routing, and JSON responses.

11. my_corporate.py  
An extended version of corporate_simple.py with improved data handling and operations.

12. positive_negative_checker.py  
A small script that checks whether a number is positive, negative, or zero.

13. sms_automation.py  
Automates sending SMS messages using Python and external SMS APIs or modules.

14. text_to_speech.py  
Converts any string or text into speech using the pyttsx3 TTS engine.

15. unit_coneverter.py  
Converts values between units such as kilometers ↔ miles, Celsius ↔ Fahrenheit, etc.

16. whatsapp_automation.py  
Automates sending WhatsApp messages using pywhatkit.


  **Technologies Used**
- Python 3  
- pyttsx3  
- speech_recognition  
- pywhatkit  
- Flask  
- smtplib  
- Standard Python libraries
